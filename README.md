# Sistema-de-control-de-nivel-y-llenado
Sistema para controlar nivel de tanque y llenado de botellas.

## Directorios
```
/firmware        (Arduino / ESP32)
/labview         (VI)
/backend         (Node-RED / server)
/docs            (reportes, diagramas)
/hardware        (diagramas eléctricos)
/test            (pruebas)
```
